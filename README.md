## 🛠 Troubleshooting

If the app does not run or crashes:

- Make sure your Windows is up to date.
- Check that you downloaded the correct file version.
- Close other applications to free up memory.
- Restart your computer and try running the app again.

If you cannot connect to a server:

- Confirm your internet connection is working.
- Try switching to a different server from the list.
- Some servers may be temporarily offline due to maintenance.
